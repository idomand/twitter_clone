# twitter clone

## Description
A micro-blogging platform (twitter clone) created using react and Firebase as an API and backend.

The platform allows for user signup using their google account or with an Email and password, uploading massages and viewing other people messages. 
## Usage

Link to live demo: https://ido-react-micro-blogging.firebaseapp.com/


### to run on your own computer

After cloning the repo, 

open the terminal and run
```bash
yarn install
```

after that, opening the code in your browser of choice will result in this image:

![](img/Stock_Exchange_Search_Engine.PNG)

from this point you can login the platform using your google account or using an email and password combo.
